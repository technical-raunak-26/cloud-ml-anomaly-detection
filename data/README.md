# Dataset Folder

This folder is intended to contain dataset-related references and preprocessing scripts.

The project uses the publicly available Intrusion Detection Evaluation Dataset (CIC-IDS2017).

Dataset Source:
https://www.unb.ca/cic/datasets/ids-2017.html

Note: The dataset files are not uploaded directly to this repository due to large size constraints. Users can download the dataset from the official source and place it in this directory before running the code.
